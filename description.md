**1 - Просмотр списка команд**  
GET:

    localhost:8088/teams
-----------------------------------------------------------------------------
**2 - Поиск команды по id**  
GET:

    localhost:8088/teams/{id}
-----------------------------------------------------------------------------
**3 - Создание команды**

    localhost:8088/teams
POST:

    {
        "teamName": "Liverpool"
    }
-----------------------------------------------------------------------------
**4 - Редактирование команды**

    localhost:8088/teams
PUT:

    {
        "teamId": {id},
        "teamName": ""
    }
-----------------------------------------------------------------------------
**5 - Удаление команды по id**  
DELETE:

    localhost:8088/teams/{id}
